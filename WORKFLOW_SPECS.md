# End-to-End Pre-Production Workflow Definition

## Phase 1: Input & Intelligence Gathering
1.  **Input:** User provides a raw brief, a product name, or a trending hashtag.
2.  **Context Injection:** System fetches "Phaya Thai LLM" weights to understand current Thai cultural sentiment.
3.  **Output:** 3 Concept Options (Creative Hook, Emotional Angle, or Comedic Angle).

## Phase 2: Scripting & Nuance Refinement
1.  **Selection:** User selects a concept.
2.  **Generation:** AI populates a 2-column script (Visual/Audio).
3.  **Nuance Check:**
    *   *Action:* User highlights a sentence.
    *   *AI suggestion:* Provides variations in *Kham-Su-Phap* (Polite) vs *Kham-Phuen* (Slang).
4.  **Formatting:** Auto-apply Thai production standards (font sizes, spacing).

## Phase 3: Visual Synthesis
1.  **Parsing:** AI extracts "Visual Descriptions" from the script's visual column.
2.  **Prompt Engineering:** System automatically appends Thai-specific tokens (e.g., "Thai cinematic lighting," "Modern Bangkok streetscape").
3.  **Rendering:** Generation of 4 keyframes per scene.
4.  **Consistency:** User locks a "Character Seed" to ensure the same actor appears throughout the storyboard.

## Phase 4: Production Breakdown (Logistics)
1.  **Entity Extraction:** AI identifies nouns in the script.
    *   *Example:* "Table with Som Tum" -> Category: Prop.
    *   *Example:* "Siam Square" -> Category: Location.
2.  **Table Generation:** A structured breakdown for the Assistant Director (AD).
3.  **Export:** Final package generation (Script + Storyboard + Breakdown).

## Phase 5: Feedback Loop
1.  **Sharing:** Generate a secure "Share Link" for clients.
2.  **Annotation:** Clients leave comments directly on script lines or storyboard frames.
3.  **Iteration:** AI re-generates specific sections based on client feedback.
