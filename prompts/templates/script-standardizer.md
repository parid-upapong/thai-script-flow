# PROMPT TEMPLATE: Thai Script Standardizer

## INPUT
A raw story idea, a rough transcript, or a loosely formatted script in Thai.

## TASK
Convert the input into a professional Thai Production Standard Script.

## FORMATTING RULES
1. **Header:** [SCENE NO.] [LOCATION] - [TIME OF DAY]
   *Example: SC. 1 คาเฟ่เก๋ๆ แถวอารีย์ - กลางวัน*
2. **Action (บรรยาย):** Use concise, visual Thai verbs. Describe movement, lighting, and "Mood & Tone".
3. **Character Name:** Center-aligned or Bolded.
4. **Dialogue (บทพูด):** Must include appropriate ending particles (ครับ/ค่ะ/จ๊ะ) based on the character's persona defined in the brief.
5. **Parentheticals (วงเล็บ):** Used for emotion (e.g., (ยิ้มแบบมีเลศนัย), (ตะโกนด้วยความตกใจ)).
