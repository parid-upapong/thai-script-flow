# Sitemap: ThaiCreative AI Platform

## 1. Authentication & Onboarding
*   Login / Register (Social Login: LINE, Facebook, Google)
*   User Profile Setup (Creator / Agency / Production House)
*   Workspace Selection

## 2. Main Dashboard
*   **Project Gallery:** Grid/List view of active productions.
*   **Trending Now:** Real-time Thai social media trends for inspiration.
*   **Quick Start:** One-click templates (TikTok, TVC, Short Film).

## 3. Creative Studio (The Workspace)
*   **Step 1: Concept & Brief**
    *   AI Brainstorming Board
    *   Trend Integration Module
*   **Step 2: Scripting Engine**
    *   Multi-format Editor (Split screen: Script/AI Chat)
    *   Dialect & Tone Tuner
    *   Collaborative Comments
*   **Step 3: Visual Storyboard**
    *   Scene Breakdown
    *   AI Image Generator (Thai Context Optimized)
    *   Moodboard Canvas
*   **Step 4: Production Logistics**
    *   AI Breakdown (Prop, Talent, Wardrobe)
    *   Location Scout Suggestions
    *   Schedule Generator

## 4. Asset Library
*   Saved Scripts
*   Generated Image Assets
*   Brand Kits (Logos, Fonts for specific clients)

## 5. Collaboration & Export
*   Review Link (Client Approval Mode)
*   Export Formats: PDF (Script), PPTX (Storyboard), CSV (Breakdown)

## 6. Settings & Admin
*   Subscription Management
*   API Key Management
*   Team Permissions
